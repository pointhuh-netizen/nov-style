# Nov Style Engine

SillyTavern용 문체 지침 프롬프트 자동 주입 확장입니다.  
축별 문체 모듈을 선택해 결합한 지침 프롬프트를 매 생성마다 자동으로 주입합니다.

---

## 설치 방법

SillyTavern 확장은 두 경로 중 하나에 설치됩니다. 본인의 ST 버전에 맞는 경로를 사용하세요.

### 방법 1: 신규 경로 (ST 1.12+, 권장)

```bash
cd [SillyTavern 루트]/data/default-user/extensions/
git clone https://github.com/pointhuh-netizen/nov-style
```

설치 후 경로: `SillyTavern/data/default-user/extensions/nov-style/`

### 방법 2: 구버전 경로

```bash
cd [SillyTavern 루트]/public/scripts/extensions/third-party/
git clone https://github.com/pointhuh-netizen/nov-style
```

설치 후 경로: `SillyTavern/public/scripts/extensions/third-party/nov-style/`

### 설치 후

1. SillyTavern을 재시작하거나 새로고침합니다.
2. 상단의 주사위 아이콘(Extensions) → **Nov Style Engine** 섹션이 나타납니다.

---

## 업데이트

```bash
cd [설치 경로]/nov-style
git pull
```

## 사용 방법

1. **Extensions 패널**에서 **📝 Nov Style Engine** 섹션을 찾습니다.
2. 각 축(Axis)의 헤더를 클릭해 펼칩니다.
3. 원하는 문체 모듈을 선택합니다:
   - **단일 선택 축** (시점, 서술방식, 어조, 세계시뮬레이션): 드롭다운에서 하나 선택
   - **복수 선택 축** (장르, 분위기, 배경, 특별요소, 상호작용): 체크박스로 여러 개 선택 가능
4. 하단의 **⚙️ 컨피그 설정**에서 등급/제한 설정을 선택합니다.
5. **✅ 적용** 버튼을 눌러 프롬프트를 활성화합니다.
6. **👁️ 미리보기** 버튼으로 생성될 프롬프트를 미리 확인할 수 있습니다.
7. **🔄 초기화** 버튼으로 모든 선택을 리셋합니다.

> 선택 상태는 **채팅별**로 저장됩니다. 채팅을 전환하면 해당 채팅의 설정이 자동 복원됩니다.

## 축(Axis) 설명

| 축 | 이름 | 유형 | 설명 |
|----|------|------|------|
| W | 세계 시뮬레이션 | 단일 | 세계의 기본 갈등 강도·NPC 자율성 |
| A | 시점 | 단일 | 1인칭/2인칭/3인칭/전지적/공간/사물 |
| S | 서술 방식 | 단일 | 동화체/무협체/서간체/극본체 등 |
| B | 어조·어휘 | 단일 | 건조체/서정체/구어체/냉소체 등 |
| C | 장르 | 복수 | 궁정정치/추리/동화/무협/로맨스/SF 등 |
| D | 분위기 | 복수 | 일상/불안/공포/몽환/축제/노스탤지어 등 |
| E | 배경 | 복수 | 전근대/강호/현대도시/학교/이세계 등 |
| F | 특별 요소 | 복수 | 마법/신격/변신/계약/오메가버스 등 |
| G | 상호작용 | 복수 | NPC 반응 방식, 능동/수동/대등형 등 |

## 컨피그(Config) 설명

| 컨피그 | 설명 |
|--------|------|
| 🎭 유저 캐릭터 제어 | AI가 {{user}}를 얼마나 서술하는지 제어 (미사용/RP모드/사칭모드) |
| 🔞 NSFW 등급 제어 | 콘텐츠 수위 제어 (전연령/암시/성인/무제한) |
| 💀 치명적 결과 프로토콜 | 사망·영구 상실 허용 여부 (미사용/허용) |

## 참고사항

- 이 확장은 **ST의 `setExtensionPrompt` API**를 사용합니다. ST 1.10.0 이상에서 정상 동작합니다.
- 문체 지침은 **시스템 프롬프트**로 주입되며, 기존 캐릭터 카드 설정과 함께 작동합니다.
- 축 선택이 없으면 핵심 규칙(마스터 규칙)만 주입됩니다.
